# Variational Autoencoder

## Environment

* OS: Ubuntu14.04
* Language: Python3.5.2

## Install packages

I recommend using pyenv and venv.

```
$ pyenv install 3.5.2
$ pyenv local 3.5.2
$ pyvenv venv
$ . venv/bin/activate
(venv) $ pip3 install -r requirements.txt
```

When you'd like to deactivate the environment,

```
(venv) $ deactivate
```
